# roman_to_int

Implement `roman_to_int(s)` in `solution.py`.

Convert an uppercase Roman numeral string built from the symbols
`I, V, X, L, C, D, M` to its integer value. Handle subtractive notation
(e.g. `IV` = 4, `IX` = 9, `XL` = 40, `CM` = 900). Input is always a valid
Roman numeral in the range 1..3999.
