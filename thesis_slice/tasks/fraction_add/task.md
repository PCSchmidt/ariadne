# fraction_add

Implement `add(f1, f2)` in `solution.py`.

Each fraction is a tuple `(numerator, denominator)` of integers. The denominator
is never zero but either value may be negative. Return the sum as a tuple
`(numerator, denominator)` in **canonical form**:

- reduced to lowest terms;
- the denominator is always **positive** (any sign lives in the numerator);
- zero is represented exactly as `(0, 1)`.
